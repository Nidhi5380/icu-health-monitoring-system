# icu-health-monitoring-system
ICU Health Monitoring System using Raspberry Pi
Final Year B.Tech project focused on real-time patient health monitoring using IoT.

##Overview
An IoT-based system designed to continuously monitor ICU patient vitals such as heart rate and body temperature using sensors connected to a Raspberry Pi.

## Technologies Used
- Raspberry Pi
- Python
- IoT Sensors
- Embedded Systems
